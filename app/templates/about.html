About This Project

The Resume Skill Gap Analyzer is an undergraduate research project developed for CS 4850 – Faculty Directed Research at Weber State University. It helps Computer Science students identify missing skills between their resume and a target job description using rule-based Natural Language Processing (NLP).

The tool uses spaCy for Named Entity Recognition, a curated CS skill taxonomy (73 skills across 5 categories) for dictionary-based matching, and scikit-learn TF-IDF cosine similarity for an overall match score.

How It Works

  1. Text preprocessing — cleans HTML, normalizes case, and tokenizes both documents.
  2. Skill extraction — runs spaCy NER and dictionary matching against a curated CS skill taxonomy to identify skills in each document.
  3. Gap computation — computes the set difference (skills in JD but not resume), organized by category.
  4. Similarity scoring — uses TF-IDF vectorization and cosine similarity for an overall text match score.
  5. Feedback generation — produces category-specific improvement tips based on the identified gaps.
